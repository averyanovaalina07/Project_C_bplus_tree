--- Для того чтобы скомпилировать и запустить основную программу ---
1. Команда для сборки (компиляции) проекта:
make
2. Команда для запуска готовой программы:
./bplus_tree


--- Для того чтобы запустить тесты ---
1. Сборка тестов:
make test
2. Запуск тестов:
./run_tests


# Скачать изменения из ветки main с сервера GitHub
git pull origin main
