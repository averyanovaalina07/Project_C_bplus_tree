# Реализация B+-дерева с поддержкой удаления, дискового хранения и журналирования 

Данный проект представляет собой реализацию B+-дерева — самобалансирующейся структуры данных, оптимизированной для систем с блочным доступом. Проект написан на языке C с использованием стандарта POSIX и реализует механизмы дискового хранения, журналирования и конкурентного доступа.

## Основные возможности

* **Полный набор операций:** Поддерживается поиск (Search), вставка (Insert), удаление (Delete) и диапазонные запросы (Range Query).
* **Дисковое хранение (mmap):** Узлы дерева хранятся в бинарном файле (`tree.dat`) с фиксированным размером блока в 4096 байт (стандартный размер страницы). Вместо указателей оперативной памяти используются индексы дисковых блоков.
* **Write-Ahead Logging (WAL):** Все операции модификации дерева предварительно записываются в лог-файл (`wal.log`) с принудительным сбросом буферов на диск (`fsync`), что гарантирует консистентность и возможность восстановления после сбоев.
* **Многопоточность и конкурентный доступ:** Защита узлов дерева реализуется с помощью блокировок чтения-записи (`pthread_rwlock_t`), что обеспечивает параллельный доступ. Запись в журнал WAL защищена мьютексами (`pthread_mutex_t`) для атомарности.
* **Настраиваемый порядок:** Порядок дерева $t \ge 2$ задается динамически при инициализации.

## Архитектура проекта

Проект разделен на модули для удобства сопровождения и тестирования:

* `bplus_tree.h` / `bplus_tree.c` — Логика алгоритмов B+-дерева (поиск, вставка, удаление, связывание листьев в двусвязный список). Все значения хранятся только в листовых узлах.
* `wal.h` / `wal.c` — Дисковый менеджер. Отвечает за инициализацию хранилища, `mmap` файла данных в память и ведение WAL-журнала.
* `main.c` — Демонстрационная программа, выполняющая пользовательский сценарий.
* `tests.c` — Изолированные модульные тесты базовых операций с использованием макроса `assert`.
* `Makefile` — Система сборки проекта (компиляция и линковка).

## Сборка и запуск

Проект использует систему сборки `Make` и компилятор `gcc`. 

Для того чтобы скомпилировать и запустить основную программу выполните:
```bash
make
./bplus_tree
```
В процессе работы программа создаст файлы tree.dat и wal.log, вставит тестовые пары ключей-значений (32-битные целые числа
), выполнит поиск, диапазонный запрос и удаление.

Для запуска тестов выполните:
```bash
make test
./run_tests
```

Чтобы удалить все объектные файлы, исполняемые бинарники и файлы баз данных (*.dat, *.log), используйте команду:
```bash
make clean
```
